# 🤖 Chatbot com IA

Um chatbot desenvolvido em **Python** que utiliza **Inteligência Artificial Generativa** para responder perguntas em linguagem natural. O projeto possui uma interface simples, permitindo que o usuário interaja facilmente com o modelo de IA.

## 🚀 Funcionalidades

* Conversa em linguagem natural
* Integração com um modelo de IA generativa
* Interface simples e intuitiva
* Respostas rápidas e contextualizadas
* Estrutura organizada para futuras melhorias

## 🛠️ Tecnologias Utilizadas

* Python
* API da OpenAI
* Streamlit
* Bibliotecas para manipulação de ambiente e requisições
* Interface gráfica/web (caso utilizada no projeto)
